# Bitcoin Portfolio AI

Production-ready AI-powered Bitcoin L1 wallet analytics.

## Features
- REST API server
- CLI mode
- AI-powered wallet analysis
- Docker support
- Rate limiting
- Secure env handling

## Install
npm install

## Run Server
npm run dev

## Run CLI
npm run dev:cli <bitcoin_address>

## Docker
docker build -t bitcoin-ai .
docker run -p 3000:3000 bitcoin-ai
